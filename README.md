# warframe-things

## Development Prerequisites
- [NPM](https://blog.teamtreehouse.com/install-node-js-npm-windows) or [Yarn](https://yarnpkg.com/lang/en/docs/install/#windows-stable)

## Setup
- NPM : `npm install`
- YARN : `yarn`

## Run project
- NPM : `npm run 1`
- YARN : `yarn 1`
  
## Build project
- NPM : `npm run 1`
- YARN : `yarn 1`
